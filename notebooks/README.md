# Notebooks

This folder contains one module folder per tutorial area.

Use Jupyter notebooks for Python workflows and Quarto `.qmd` files for R-heavy
workflows. The R documents can still live under `notebooks/` because they are
the runnable tutorial artifacts for those modules.

Recommended order for the first tutorial pass:

1. `miaproc_biomass`
2. `miaproc_eddy`
3. `geecomposer`
4. `mia_climate`
5. `mia_lidar`
6. `mia_allometric`
7. `mia_occupancy`
